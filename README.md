# Contact Manager (Python + SQLite3)
This is my first successful Python experience! A simple command-line Contact Manager application connected to a SQLite3 database.

# Features
- Add new contacts
- Search contacts
- Edit existing contacts
- Delete contacts
- Count total contacts
- Interactive keypad interface
- Persistent storage using SQLite3

# Technologies Used
- Python
- SQLite3

# How to Run
- Ensure Python 3 is installed.
- Download or clone the repository.
- Navigate to the project directory.
- Run:
python main.py
